# 👑 PRIME Bot

**PRIME** — бот для модерации Telegram-чатов с системой РП-уровней.

## 🎯 Возможности

- 🎲 Игры (кости, угадай число)
- 📖 Случайные фразы из истории чата (каждые 5-15 минут)
- 👑 10 уровней РП-админов
- ⚠️ Система варнов (5 = бан)
- 🐢 Медленный режим
- 📊 Статистика чата

## 📌 Команды

### Для всех
| Команда | Описание |
|---------|----------|
| `/dice` | Кости |
| `/guess 1-10` | Угадать число |
| `/myrole` | Ваш уровень |
| `/profile` | Ваш профиль |
| `/top` | Топ активных |
| `/stats` | Статистика |
| `/rp_admins` | Список РП-админов |

### Модерация
| Уровень | Команда |
|---------|---------|
| 2 | `/warns @user` |
| 3 | `/slowmode <сек>` |
| 4 | `/warn @user <причина>` |
| 5 | `/unwarn @user` |
| 6 | `/mute @user`, `/unmute @user` |
| 7 | `/kick @user` |
| 8 | `/ban @user`, `/unban @user` |
| 9 | `/clear <кол-во>` |
| 10 | `/set_rp @user <1-10>` |
